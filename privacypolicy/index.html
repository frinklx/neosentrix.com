Privacy Policy for Intrabot

1. Data Collection

Intrabot collects and processes certain data to provide its services. This includes:

  • Server IDs
  • Channel IDs
  • User IDs
  • Message content (for command processing)
  • Server member information
  • Server roles and permissions

// Note: We use Discord's Privileged Intents (GUILD_MEMBERS, GUILD_PRESENCES, MESSAGE_CONTENT)

2. Use of Data

The collected data is used for:

  • Command execution and response
  • Server management features
  • Customization of bot functionality
  • Analytics to improve bot performance

3. Data Storage

All data is stored securely and is only accessible to authorized bot developers. We use industry-standard security measures to protect your data.

4. Data Retention

Data is retained only as long as necessary for bot functionality. Users can request data deletion at any time.

5. User Rights

Users have the right to:

  • Request data deletion
  • Access their stored data
  • Opt-out of data collection
  • Request data corrections
